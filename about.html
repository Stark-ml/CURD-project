About This Project

This project is a simple CRUD application built using Vanilla JavaScript and the JSONPlaceholder API.

It demonstrates:

  • Fetching data using Fetch API
  • CRUD operations (Create, Read, Update, Delete)
  • LocalStorage usage
  • Pagination with Load More button
  • Clean UI structure (Header / Main / Footer)

This project is intended for learning and practice purposes.
